## Prerequisites

* [Python 3.10](https://www.python.org/)

## Installation

1. Download [Python 3.10](https://www.python.org/)

2. Clone the repo
   ```
   git clone https://github.com/Theyka/proxy-scraper-flask-app.git
   ```
3. Install python libraries
   ```
   pip install -r requirements.txt
   ```
4. Start python code
   ```
   python app.py
   ```
